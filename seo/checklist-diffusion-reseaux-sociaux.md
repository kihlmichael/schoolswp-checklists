# Checklist — Diffusion réseau social (post-publication)

Article : [WiseWand — Avis rédaction SEO](https://schoolswp.com/wisewand-avis-redaction-seo/)

## 1) Préparation des assets

- [x] Titre accrocheur adapté à chaque plateforme
- [x] 3 angles de posts différents (avis, résultat, comparaison)
- [ ] Visuel principal (1200×630 pour Facebook/LinkedIn, 1080×1080 pour Insta)
- [ ] Visuel Stories / Réels (1080×1920) si pertinent
- [x] Extrait clé ou citation marquante de l'article
- [x] URL trackée (UTM : source, medium, campaign)

## 2) LinkedIn

- [x] Post texte : hook → insight → CTA vers l'article
- [x] Format : 1 idée forte, pas de lien dans le corps (lien en commentaire)
- [x] Hashtags : 3 à 5 max (#SEO #IA #Rédaction #WordPress #WiseWand)
- [x] Mentionner les personnes / outils cités si pertinent
- [x] Republier en commentaire avec le lien direct
- [x] Partager dans 1–2 groupes LinkedIn pertinents

## 3) X (Twitter)

- [x] Tweet principal : hook court + bénéfice + lien
- [x] Thread optionnel (3 à 5 tweets) : résumé des points clés
- [x] Hashtags : 1 à 3 max
- [x] Taguer les comptes mentionnés dans l'article (@WiseWand)
- [ ] Épingler le tweet si c'est un article stratégique

## 4) Facebook

- [x] Post avec contexte personnel ou retour d'expérience
- [ ] Visuel ou capture d'écran de l'article
- [x] Partage dans 1–3 groupes Facebook ciblés (WordPress, SEO, IA)
- [ ] Répondre aux premiers commentaires dans les 2h

## 5) Autres canaux

- [x] Newsletter : inclure l'article dans le prochain envoi
- [x] Email dédié si article stratégique
- [x] Pinterest : épingle avec visuel vertical + description SEO
- [x] Communauté Discord / Slack si pertinent
- [ ] Republication sur Medium / dev.to si audience cible présente

## 6) Suivi post-diffusion (48–72h)

- [ ] Vérifier les stats UTM (clics, sources)
- [ ] Répondre à tous les commentaires / messages
- [ ] Identifier les posts qui performent → booster ou recycler
- [x] Programmer un re-partage à J+7 avec un angle différent
- [ ] Noter les enseignements pour la prochaine diffusion
