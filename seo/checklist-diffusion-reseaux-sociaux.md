# Checklist — Diffusion réseau social (post-publication)

Article : [WiseWand — Avis rédaction SEO](https://schoolswp.com/wisewand-avis-redaction-seo/)

## 1) Préparation des assets

- [ ] Titre accrocheur adapté à chaque plateforme
- [ ] 3 angles de posts différents (avis, résultat, comparaison)
- [ ] Visuel principal (1200×630 pour Facebook/LinkedIn, 1080×1080 pour Insta)
- [ ] Visuel Stories / Réels (1080×1920) si pertinent
- [ ] Extrait clé ou citation marquante de l'article
- [ ] URL trackée (UTM : source, medium, campaign)

## 2) LinkedIn

- [ ] Post texte : hook → insight → CTA vers l'article
- [ ] Format : 1 idée forte, pas de lien dans le corps (lien en commentaire)
- [ ] Hashtags : 3 à 5 max (#SEO #IA #Rédaction #WordPress #WiseWand)
- [ ] Mentionner les personnes / outils cités si pertinent
- [ ] Republier en commentaire avec le lien direct
- [ ] Partager dans 1–2 groupes LinkedIn pertinents

## 3) X (Twitter)

- [ ] Tweet principal : hook court + bénéfice + lien
- [ ] Thread optionnel (3 à 5 tweets) : résumé des points clés
- [ ] Hashtags : 1 à 3 max
- [ ] Taguer les comptes mentionnés dans l'article (@WiseWand)
- [ ] Épingler le tweet si c'est un article stratégique

## 4) Facebook

- [ ] Post avec contexte personnel ou retour d'expérience
- [ ] Visuel ou capture d'écran de l'article
- [ ] Partage dans 1–3 groupes Facebook ciblés (WordPress, SEO, IA)
- [ ] Répondre aux premiers commentaires dans les 2h

## 5) Autres canaux

- [ ] Newsletter : inclure l'article dans le prochain envoi
- [ ] Email dédié si article stratégique
- [ ] Pinterest : épingle avec visuel vertical + description SEO
- [ ] Communauté Discord / Slack si pertinent
- [ ] Republication sur Medium / dev.to si audience cible présente

## 6) Suivi post-diffusion (48–72h)

- [ ] Vérifier les stats UTM (clics, sources)
- [ ] Répondre à tous les commentaires / messages
- [ ] Identifier les posts qui performent → booster ou recycler
- [ ] Programmer un re-partage à J+7 avec un angle différent
- [ ] Noter les enseignements pour la prochaine diffusion
