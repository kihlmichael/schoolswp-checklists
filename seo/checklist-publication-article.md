# Checklist — Publication d’article SEO (rank-ready)

## 1) Avant d’écrire
- [ ] Intention claire (info / comparatif / tuto / achat)
- [ ] Mot-clé principal + 3 variantes
- [ ] Angle unique (ce que les autres ne font pas)
- [ ] Plan H2/H3 validé
- [ ] Pages internes à pousser (2 à 5) identifiées

## 2) Pendant l’écriture
- [ ] H1 = promesse claire
- [ ] Intro : problème → promesse → plan
- [ ] 1 idée = 1 section
- [ ] Blocs scannables (listes, étapes, tableaux si utile)
- [ ] Exemples concrets / captures / configs
- [ ] FAQ (3 à 6 questions)

## 3) Optimisation SEO
- [ ] Title SEO orienté bénéfice
- [ ] Meta description (bénéfice + preuve + CTA)
- [ ] URL courte, propre
- [ ] 1 image “hero” + alt descriptif
- [ ] Liens internes : 3 à 8 (dont 1–2 pages business)
- [ ] 1–3 liens sortants de référence
- [ ] Schema (FAQ si pertinent)

## 4) Publication
- [ ] Catégorie + tags cohérents
- [ ] CTA principal (newsletter / ressource)
- [ ] CTA secondaire (outil / formation / audit)
- [ ] Vérif mobile

## 5) Après publication (48–72h)
- [ ] Soumission indexation (si utilisé)
- [ ] Ajout dans le maillage (pages stratégiques)
- [ ] 1 post LinkedIn / 1 email / 1 repurpose
- [ ] Note “MAJ prévue” (30 jours)
