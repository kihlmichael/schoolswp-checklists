# Checklist — Maintenance mensuelle WordPress

> Objectif : garder un site rapide, sûr et à jour, chaque mois.
> Pour qui : toute personne qui gère un ou plusieurs sites WordPress.

## 1) Sauvegardes
- [ ] Sauvegarde complète testée (restauration réelle, pas juste le fichier)
- [ ] Sauvegarde stockée hors-site (cloud / externe)

## 2) Mises à jour (sur staging d'abord)
- [ ] Cœur WordPress
- [ ] Thème
- [ ] Extensions (une par une si site critique)
- [ ] Vérif visuelle + fonctionnelle après update

## 3) Sécurité
- [ ] Scan de sécurité (malware / fichiers modifiés)
- [ ] Revue des comptes utilisateurs (rôles, comptes inactifs)
- [ ] Tentatives de connexion suspectes contrôlées

## 4) Performance
- [ ] Cache purgé et reconstruit
- [ ] Base de données nettoyée (révisions, transients, spam)
- [ ] Score PageSpeed mobile + desktop relevé
- [ ] Images lourdes récentes optimisées

## 5) SEO & contenu
- [ ] Search Console : erreurs d'indexation
- [ ] Liens cassés (interne + externe)
- [ ] 1 contenu mis à jour (article qui décroche)

## 6) Reporting
- [ ] Rapport mensuel (trafic, perf, actions faites)
- [ ] Liste des points à traiter le mois suivant
